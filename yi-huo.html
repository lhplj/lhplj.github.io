异或^

lhplj

异或运算符^运算规则:相同为0,不同为1

0 ^ 0 = 0;
1 ^ 1 = 0;
0 ^ 1 = 1;
1 ^ 0 = 1;

则对于任意整数N满足

N ^ N = 0;
0 ^ N = N;

进一步,对于任意整数M和N满足

M ^ N ^ N = M;
N ^ M ^ N = M;

使用场景

  1. 数值交换
    # 交换 a 与 b 的值
int a = 7;
int b = 9;
# 正常数值交换,需要使用一个临时变量
int temp = a;
a = b;
b = temp;

# 通过异或
# a = 7 ^ 9;
a = a ^ b;
# b = 7 ^ 9 ^ 9 则 b = 7
b = a ^ b;
# a = 7 ^ 9 ^ 7 则 a = 9 完成交换
a = a ^ b;

# 注意事项: 通过异或交换数值时,要保证2个数指向的不是同一个地址
  2. 一个数组中只有一种数有奇数个,找到这个数
    # 对于数组 arr 中只有一种数有奇数个,其它为偶数个
int eor = 0;
for (int i = 0; i < arr.length; i++) {
    eor ^= arr[i];
}
# 最后 eor 就是要求的值
